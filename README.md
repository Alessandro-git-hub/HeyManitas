# HeyManitas Marketplace

A comprehensive platform connecting customers with skilled service professionals.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

✨ Features
For Customers
Browse Services: Discover skilled professionals by category and location
Smart Booking: Book services with real-time availability
Secure Payments: Multiple payment options with secure processing
Reviews & Ratings: Make informed decisions with community feedback
Order Tracking: Real-time updates on service progress
For Service Providers
Professional Profiles: Showcase skills, experience, and portfolio
Booking Management: Accept/decline bookings with calendar integration
Earnings Dashboard: Track income and performance metrics
Customer Communication: Built-in messaging system
Flexible Scheduling: Set availability and manage workload
Platform Features
Real-time Updates: Live notifications and status changes
Mobile Responsive: Optimized for all devices
Search & Filters: Advanced filtering by location, price, and ratings
Secure Authentication: Firebase-powered user management
Multi-category Support: Cleaning, tutoring, fitness, and more
🛠 Tech Stack
Frontend: React 18, Vite, Tailwind CSS
Backend: Firebase (Firestore, Authentication, Storage)
Deployment: Vercel/Netlify ready
State Management: React Context API
UI Components: Custom component library
Real-time: Firebase real-time database

📁 Project Structure
src/
├── components/          # Reusable UI components
├── pages/              # Main application pages
├── contexts/           # React Context providers
├── hooks/              # Custom React hooks
├── utils/              # Utility functions
└── styles/             # Global styles and Tailwind config
